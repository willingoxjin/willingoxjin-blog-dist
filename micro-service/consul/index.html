注册中心 Consul

# 注册中心Consul

Spring Cloud 中大部分组件都有备选方案，例如注册中心，除了常见 Eureka 之外，还有 Zookeeper、 Consul、Nacos。

Consul 是 HashiCorp 公司推出来的开源产品。主要提供服务发现、服务隔离、服务配置等功能。

相比于 Eureka 和 zookeeper，Consul 配置更加一站式，因为它内置了很多微服务常见的需求:服务发现与注册、分布式一致性协议实现、健康检查、键值对存储、多数据中心等。

Consul官网：https://www.consul.io/docs/intro

# Consul安装

建议直接去官网：https://www.consul.io/downloads

这里列举两种方式：

# Mac

使用brew安装

brew tap hashicorp/tap
brew install hashicorp/tap/consul

# Consul 服务消费

Consul 服务消费和Eureka是一样的，使用LoadBalancerClient可以获取服务实例，使用RestTemplate发起HTTP请求进行服务调用

# Consul 服务提供者集群

服务名相同的多个服务提供者会被视为相同的副本，只要注册到Consul，在调用时会根据负载均衡策略选择调用哪个服务
