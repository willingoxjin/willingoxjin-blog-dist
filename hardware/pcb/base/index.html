# 二、PCB 介绍

# 1.什么是 PCB ？

PCB（Printed Circuit Board）中文叫印刷电路板，或叫印刷线路板。顾名思义，PCB 就如同像纸张印刷一样被印刷出来。

电路板在我们日常生活中无处不在，如电视遥控器、电子手表、手机、电脑中都使用到了 PCB。

PCB 的功能类似于导线，将各个元器件连接起来，并且固定元器件。

# 三、PCB 专业术语

# 1.PCB 多层板

单层板：又称单面板，在基板的一面覆盖有导电的铜皮作为线路层，而另一面通常没有线路。阻焊层和丝印层则分别用于保护线路和标识元件信息。

  • 铜皮：导线的作用。
  • 基板（绝缘板）
  • 阻焊层（绿油）：用于防止触电、保护电路不被空气氧化。
  • 丝印层：用于标注元器件。

PCB 双层板

如果电路复杂，会有导线交叉的情况，单层板就无法满足需求，于是就有了双层板。

  • 过孔：双层板中的过孔，是用于连接 PCB 不同层之间线路的导通孔。
  • 顶层：即双层板的上层，用于布置较为关键或主要的电路线路。
  • 底层：用于布置其他辅助线路或相对次要的线路。

PCB 四层板

四层板相当于两个双层板叠在一起，并在两个双层板之间加一个芯板。

  • 芯板实际上就是是绝缘板。

PCB 六层板：类比四层板，在四层板的基础上，在加一个双层板。

当然继续类比还有八层板、十层板等等。
