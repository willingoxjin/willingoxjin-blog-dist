# SVG 简介

什么是SVG？

  • SVG 指可伸缩矢量图形 (Scalable Vector Graphics)
  • SVG 用来定义用于网络的基于矢量的图形
  • SVG 使用 XML 格式定义图形
  • SVG 图像在放大或改变尺寸的情况下其图形质量不会有所损失
  • SVG 是万维网联盟的标准
  • SVG 与诸如 DOM 和 XSL 之类的 W3C 标准是一个整体

SVG 是 W3C 推荐标准：SVG 于 2003 年 1 月 14 日成为 W3C 推荐标准。

# SVG坐标系

x=0, y=0 点在左上角，向右x值增加，向下y值增加

# SVG 预定义的形状

SVG有一些预定义的形状元素，可被开发者使用和操作：

  • 矩形 <rect>
  • 圆形 <circle>
  • 椭圆 <ellipse>
  • 线 <line>
  • 折线 <polyline>
  • 多边形 <polygon>
  • 路径<path>

# <path>的属性介绍

<path> 元素用于定义一个路径。使用 Path 属性 ，你可以自己画圆，椭圆，三角形、多边形等等多种多样的图形

属性：

  • d="定义路径指令"
  • pathLength="如果存在，路径将进行缩放，以便计算各点相当于此值的路径长度"
  • transform="转换列表"
  • 显现属性: Color, FillStroke, Graphics, Markers

# d属性

下面的命令可用于路径数据：

  • M moveto(M X,Y) ：起点

M 0 100：大写M表示从坐标(0, 100) 开始画图； m -75 0：小写m表示相对坐标，即相对M 0 100 的坐标

如 M 120, 50 m -80, 0 代表以坐标 (120, 50) 为原点，偏移到坐标 (-80, 0)

  • L ineto(L X,Y) ：画线

指定线到达的坐标点

如 d="M 0 100 L 100 100"

  • H horizontal lineto(H X)：画一条水平的线

  • V vertical lineto(V Y)：画一条垂直的线

  • C curveto(C X1,Y1,X2,Y2,ENDX,ENDY)：三次贝赛曲线

  • S smooth curveto(S X2,Y2,ENDX,ENDY)：画一条光滑的曲线

  • Q quadratic Belzier curve(Q X,Y,ENDX,ENDY)：画一条二次贝塞尔曲线

  • T smooth quadratic Belzier curveto(T ENDX,ENDY)：画一条光滑的二次贝塞尔曲线

  • A elliptical Arc(A RX,RY,XROTATION,FLAG1,FLAG2,X,Y)： 画椭圆的弧，你可以根据此方式画圆

语法：A rx ry x-axis-rotation large-arc-flag sweep-flag X Y

rx：椭圆横轴半径

ry：椭圆竖轴半径

x-axis-rotation：椭圆横轴相对于 Canvas X 轴的偏移角度

large-arc-flag：在前面三个参数确定的情况下，满足当前点到指定点 (X, Y) 位置条件的圆弧总是有四条，此值取0表示绘制小弧度，取值1表示绘制大弧度

sweep-flag：在前三个参数确定的情况下，满足当前点到指定点(X,Y)位置条件的圆弧总是有四条，去掉上面 large-arc-flag 标识后还有两个，sweep-flag 取值 0 表示绘制逆时针方向的圆弧，取值1表示绘制顺时针方向的圆弧

X 圆弧需要经过的坐标 x

Y 圆弧需要经过的坐标 y

演示图如下：

  • Z closepath() 关闭路径

WARNING

以上所有命令均允许小写字母。大写表示绝对定位，小写表示相对定位。

# 示例—线

来看一个SVG使用path绘制圆的示例图：

画一个等腰三角形

<svg xmlns="http://www.w3.org/2000/svg" version="1.1" height="110" width="110">
    <!-- 画线 从(0, 100)到(100,100) -->
    <path d="M 0 100 L 100 100" stroke="red" stroke-width="3" fill="none" />
    <!-- 画线 从(0, 100)到(50,20) -->    
    <path d="M 0 100 L 50 20" stroke="red" stroke-width="3" fill="none" />
    <!-- 画线 从(50, 20)到(100,100) -->
    <path d="M 50 20 L 100 100" stroke="red" stroke-width="3" fill="none" />
</svg>

# 一次性画多条线

事实上，你可以一次画多条线，如下

<svg xmlns="http://www.w3.org/2000/svg" version="1.1" height="110" width="110">
	<path d="M 0 100 
             L 100 100 
             L 50 20 
             L 0 100" 
          stroke="red" 
          stroke-width="3" 
          fill="none" />
</svg>

TIPS

在一次性画完所有的点的时候，每次画点的起点，都是上一个点的终点

如上面的示例

  • 起点 0 100 终点 100 100
  • 起点 100 100 终点 50 20
  • 起点 50 20 终点 0 100 （最终回到原点）

你也可以使用 Z 属性简化最后一步来封闭三角形，Z 会直接将画笔所在坐标点与起点坐标之间画一条线段

<svg xmlns="http://www.w3.org/2000/svg" version="1.1" height="110" width="110">
	<path d="M 0 100 
             L 100 100 
             L 50 20 
             Z" 
          stroke="red" 
          stroke-width="3" 
          fill="none" />
</svg>

    <path d="M 0 100 l 50 -80" stroke="red" stroke-width="3" fill="none" />

stroke 属性：

  • stroke 轮廓颜色
  • stroke-width 轮廓的厚度
  • stroke-linecap 在画线的尾部的结点样式，属性： butt round-圆角 square
  • stroke-dasharray 创建虚线 其参数为，实线距离、虚线距离相交替

fill 代表填充的颜色 none 代表不填充
