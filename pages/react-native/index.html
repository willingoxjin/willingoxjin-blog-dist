# IOS

在Mac平台上开发 React Native 需要安装以下环境和工具：

  • Node & Watchman
  • XCode & CocoaPods

安装Node & Watchman

React Native 开发需要用到 Node.js 环境。我们做 React Native 开发会经常性的和 Node.js 进行打交道，比如：我们用 npm start 命令启动 React Native 的启动器；用 npm install 安装项目所依赖的第三方组件；用 npm publish 往 npm 上发布一些组件等。

Note.js 基于 Chrome’s V8 JavaScript 引擎。Node.js是一个事件驱动I/O服务端JavaScript环境。

打开终端运行如下命名：

brew install node
brew install watchman
1
2

当然你也可以不使用 brew 安装，直接到 Note.js (opens new window)官网下载安装包进行安装。

安装iOS开发工具XCode（IOS）

React Native 目前需要Xcode (opens new window) 12 或更高版本。你可以通过 App Store 或是到Apple 开发者官网 (opens new window)上下载。这一步骤会同时安装 Xcode IDE、Xcode 的命令行工具和 iOS 模拟器。

在 Xcode 中安装 iOS 模拟器：安装模拟器只需打开 Xcode > Preferences... 菜单，然后选择 Components 选项，即可看到各种可供安装的不同的 iOS 版本的模拟器。

安装CocoaPods （IOS）

CocoaPods (opens new window)是用 Ruby 编写的包管理器（可以理解为针对 iOS 的 npm）。从 0.60 版本开始 react native 的 iOS 版本需要使用 CocoaPods 来管理依赖。你可以使用下面的命令来安装 CocoaPods。CocoaPods的版本需要 1.10 以上。

sudo gem install cocoapods
## or
brew install cocoapods

# 修改npm镜像，提高项目初始化的速度

我们在初始化React Native应用或从npm上安装一些组件的时候通常的情况下是比较慢的，这是因为npm的服务是设在国外的，所以在国内访问的速度不是很理想。提高项目初始化的速度我们可以为npm设置一个国内镜像，让npm每次下载组件的时候都从国内的镜像上获取这样一来速度就会大大提高。

iOS修改npm镜像的方法：

在MAC电脑上我们可以在.npmrc文件中设置npm的下载镜像地址，.npmrc文件通常在/Users/用户名/.npmrc路径下，打开此文件然后添加：

registry = https://registry.npm.taobao.org
1

如果没有修改权限，那么可以将此文件复制一份到别的地方，修改完成之后再替换过去即可。

这样一来访问npm的时候就会被重定向到我们所设置的https://registry.npm.taobao.org镜像服务器，所以说速度就会大大提高。

# 主导航器（APP中的页面）

import {createAppContainer, createSwitchNavigator} from 'react-navigation';
import {createStackNavigator} from 'react-navigation-stack';

// 初始化的导航器
const InitNavigator = createStackNavigator({
    // 欢迎页
    WelcomePage: {
        screen: WelcomePage, // screen 渲染的组件页面
        // 导航器参数
        navigationOptions: {
            // 是否显示头部导航
            headerShown: false,// 可以通过将header设为null 来禁用StackNavigator的Navigation Bar
        },
    },
});
const MainNavigator = createStackNavigator({
    HomePage: {
        screen: HomePage,
        navigationOptions: {
            headerShown: false,
        },
    },
    DetailPage: {
        screen: DetailPage,
        navigationOptions: {
            headerShown: false,
        },
    },
    WebViewPage: {
        screen: WebViewPage,
        navigationOptions: {
            headerShown: false,
        },
    },
}, {
    // 默认导航器参数
    defaultNavigationOptions: {
        headerShown: false,
    },
});
// 创建 app 导航器的内部容器页面
export default createAppContainer(createSwitchNavigator({
    Init: InitNavigator,
    Main: MainNavigator,
}, {
    navigationOptions: {
        headerShown: false,
    },
}));

# 底部导航器

import React, {Component} from 'react';
import {createAppContainer} from 'react-navigation';
import {createBottomTabNavigator} from 'react-navigation-tabs';
import {connect} from 'react-redux';
import HomePage from '../page/HomePage';
import MyPage from '../page/MyPage';
import WorkBench from '../page/WorkBench';
import MaterialIcons from 'react-native-vector-icons/MaterialIcons';
import Ionicons from 'react-native-vector-icons/Ionicons';
import Entypo from 'react-native-vector-icons/Entypo';
import EventTypes from '../util/EventTypes';
import {BottomTabBar} from 'react-navigation-tabs';
import EventBus from 'react-native-event-bus';

// 在这里配置页面的路由
const TABS = {
    // 首页
    HomePage: {
        screen: HomePage,
        navigationOptions: {
            tabBarLabel: '首页',
            tabBarIcon: ({tintColor, focused}) => (
                <MaterialIcons
                    name={'whatshot'}
                    size={26}
                    style={{color: tintColor}}
                />
            ),
        },
    },
    WorkBench: {
        screen: WorkBench,
        navigationOptions: {
            tabBarLabel: '工作台',
            tabBarIcon: ({tintColor, focused}) => (
                <MaterialIcons
                    name={'favorite'}
                    size={26}
                    style={{color: tintColor}}
                />
            ),
        },
    },
    // 我的页面
    MyPage: {
        screen: MyPage,
        navigationOptions: {
            tabBarLabel: '我的',
            tabBarIcon: ({tintColor, focused}) => (
                <Entypo
                    name={'user'}
                    size={26}
                    style={{color: tintColor}}
                />
            ),
        },
    },
};

class DynamicTabNavigator extends Component<Props> {
    constructor(props) {
        super(props);
        console.disableYellowBox = true;
    }

    _tabNavigator() {
        if (this.Tabs) {
            return this.Tabs;
        }
        const {HomePage, WorkBench, MyPage} = TABS;
        const tabs = {WorkBench, MyPage};// 根据需要定制显示的tab
        HomePage.navigationOptions.tabBarLabel = '首页';// 动态配置Tab属性
        // 创建导航器
        return this.Tabs = createAppContainer(createBottomTabNavigator(tabs, {
                tabBarComponent: props => {
                    return <TabBarComponent theme={this.props.theme} {...props}/>;
                },
            },
        ));
    }

    render() {
        const Tab = this._tabNavigator();
        // 渲染导航器
        // onNavigationStateChange 导航器变更回调 
        return <Tab
            onNavigationStateChange={(prevState, newState, action) => {
                EventBus.getInstance().fireEvent(EventTypes.bottom_tab_select, {//发送底部tab切换的事件
                    from: prevState.index,
                    to: newState.index,
                });
            }}
        />;
    }
}

// 底部导航器
class TabBarComponent extends React.Component {
    constructor(props) {
        super(props);
        this.theme = {
            tintColor: props.activeTintColor,
            updateTime: new Date().getTime(),
        };
    }

    render() {
        // activeTintColor 激活时的颜色
        return <BottomTabBar
            {...this.props}
            activeTintColor={this.props.theme.themeColor}
        />;
    }
}

// 从redux获取主题颜色
const mapStateToProps = state => ({
    theme: state.theme.theme,
});

export default connect(mapStateToProps)(DynamicTabNavigator);

import React, {Component} from 'react';

import NavigationUtil from '../navigator/NavigationUtil';
import SplashScreen from 'react-native-splash-screen';
import actions from '../action';
import {connect} from 'react-redux';

class WelcomePage extends Component<Props> {
    componentDidMount() {
        this.props.onThemeInit();
        // 200ms 后进入下一个路由
        this.timer = setTimeout(() => {
            SplashScreen.hide();
            // 重置页面到 HomePage
            NavigationUtil.resetToHomPage({
                navigation: this.props.navigation,
            });
        }, 200);
    }

    componentWillUnmount() {
        this.timer && clearTimeout(this.timer);
    }

    render() {
        return null;
    }
}

const mapDispatchToProps = dispatch => ({
    onThemeInit: () => dispatch(actions.onThemeInit()),
});

export default connect(null, mapDispatchToProps)(WelcomePage);

# 自定义顶部导航器：

import React, {Component} from 'react'
import {
  ViewPropTypes,
  Text,
  StatusBar,
  StyleSheet,
  View,
  Platform,
  DeviceInfo,
  TouchableOpacity, Dimensions, PixelRatio,
} from 'react-native'
import PropTypes from 'prop-types'
import NavigationUtil from '../../utils/navigationUtil'
import { SvgXml } from 'react-native-svg'
import { backLeftBtn } from '../../svg/common'

const NAV_BAR_HEIGHT_IOS = 44 // 导航栏在iOS中的高度
const NAV_BAR_HEIGHT_ANDROID = 50 // 导航栏在Android中的高度
const NAV_BAR_HEIGHT =
  Platform.OS === 'ios' ? NAV_BAR_HEIGHT_IOS : NAV_BAR_HEIGHT_ANDROID
const STATUS_BAR_HEIGHT =
  Platform.OS !== 'ios' || DeviceInfo.isIPhoneX_deprecated ? 0 : 44 // 状态栏的高度
// 设置状态栏所接受的属性
const StatusBarShape = {
  barStyle: PropTypes.oneOf(['light-content', 'default', 'dark-content']),
  hidden: PropTypes.bool,
  backgroundColor: PropTypes.string,
}

const { width: SCREEN_WIDTH } = Dimensions.get('window')
const scaleSize = size =>
  PixelRatio.roundToNearestPixel(size * (SCREEN_WIDTH / 375))

export const NAVIGATION_BAR_HEIGHT = NAV_BAR_HEIGHT + STATUS_BAR_HEIGHT
export default class NavigationBar extends Component {
  // 提供属性的类型检查
  static propTypes = {
    style: ViewPropTypes.style,
    title: PropTypes.string,
    titleView: PropTypes.element,
    titleLayoutStyle: ViewPropTypes.style,
    hide: PropTypes.bool,
    statusBar: PropTypes.shape(StatusBarShape),
    rightButton: PropTypes.element,
    leftButton: PropTypes.element,
    navigation: PropTypes.any,
  }
  // 设置默认属性
  static defaultProps = {
    statusBar: {
      barStyle: 'light-content',
      hidden: false,
    },
  }

  render() {
    // 状态栏
    let statusBar = !this.props.statusBar.hidden ? (
      <View style={styles.statusBar}>
        <StatusBar {...this.props.statusBar} />
      </View>
    ) : null

    // 标题栏
    let titleView = this.props.titleView ? (
      this.props.titleView
    ) : (
      <Text ellipsizeMode="head" numberOfLines={1} style={styles.title}>
        {this.props.title}
      </Text>
    )

    // 内容
    let content = this.props.hide ? null : (
      <View style={styles.navBar}>
        {this.props.leftButton
          ? this.getButtonElement(this.props.leftButton)
          : this.getLeftDefaultButtonElement()}
        <View
          style={[styles.navBarTitleContainer, this.props.titleLayoutStyle]}>
          {titleView}
        </View>
        {this.getButtonElement(this.props.rightButton)}
      </View>
    )
    return <View style={[styles.container, this.props.style]}>
      {statusBar}
      {content}
    </View>
  }

  getButtonElement(data) {
    return <View style={styles.navBarButton}>{data ? data : null}</View>
  }

  // 左边默认元素 backLeftBtn 为svg的图，可以更换成 icon
  getLeftDefaultButtonElement() {
    return (
      <TouchableOpacity
        style={{padding: 8, paddingLeft: 12}}
        onPress={() => NavigationUtil.goBack(this.props.navigation)}>
        <View style={styles.dLeftBtn}>
          <SvgXml xml={backLeftBtn} />
        </View>
      </TouchableOpacity>
    )
  }
}

const styles = StyleSheet.create({
  container: {
  },
  navBarButton: {
    alignItems: 'center',
  },
  navBar: {
    flexDirection: 'row',
    alignItems: 'center',
    justifyContent: 'space-between',
    height: NAV_BAR_HEIGHT,
  },
  navBarTitleContainer: {
    alignItems: 'center',
    justifyContent: 'center',
    position: 'absolute',
    left: scaleSize(44),
    right: scaleSize(44),
    top: 0,
    bottom: 0,
  },
  title: {
    fontSize: scaleSize(18),
    color: '#111315',
    fontWeight: '700',
    textAlign: 'center',
  },
  statusBar: {
    height: STATUS_BAR_HEIGHT,
  },
