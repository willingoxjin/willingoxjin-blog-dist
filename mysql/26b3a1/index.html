# 会话变量

​ 以@符号开头的变量是会话变量。直到会话结束前它可用和可访问

设置变量

-- 为变量赋值
set @hello = 'hello world!';
set @hello := 'hello world!';

-- 查询并赋值
select @hello = 'hello world!';	// 返回0
select @hello := 'hello world!'; // 返回结果

在查询语句中使用变量

 select @maxPrice:=max(price) as max_price from product;

  • 用户定义的变量不区分大小写

  • 可以将用户定义的变量分配给某些数据类型，例如整数，浮点，小数，字符串或NULL。 由一个客户端定义的用户定义的变量不被其他客户端看到。 换句话说，用户定义的变量是特定于会话的

  • 用户定义的变量只能保存一个值。 如果SELECT语句返回多个值，则变量将获取结果中最后一行的值。

  • 提取数据

    1

  • 关闭游标

    close [游标名];
    1

示例——使用游标获取指定部门的薪水和

DROP PROCEDURE IF EXISTS proc_c1;
DELIMITER $
CREATE PROCEDURE proc_c1(
	p_dno INT,
	OUT p_sal DECIMAL(18,2)
)
BEGIN
	DECLARE err INT DEFAULT 0;
	DECLARE v_sal DECIMAL(18,2) DEFAULT 0;
	DECLARE cur CURSOR FOR SELECT sal FROM empdb.emp WHERE deptno=p_dno;
	DECLARE CONTINUE HANDLER FOR NOT FOUND SET err=1;
	SET p_sal=0;
	OPEN cur;
	FETCH cur INTO v_sal;
	WHILE err=0 DO
		SET p_sal=p_sal+v_sal;
		FETCH cur INTO v_sal;
	END WHILE;
	CLOSE cur;
	-- deallocate cur;
END $
DELIMITER ;

CALL proc_c1(10,@s);
SELECT @s;

案例——数的平方

DROP FUNCTION IF EXISTS func_power;
DELIMITER $
CREATE FUNCTION func_power(
	p_num DECIMAL(18,2)
) RETURNS DECIMAL(18,2)
BEGIN
	RETURN p_num*p_num;
END $
DELIMITER ;

SELECT func_power(2); -- 4

# Mysql中函数和存储过程的区别

存储过程：

  • 可以写sql语句

  • inout，out构造返回值

  • 调用：call：存储过程名称

  • 可以返回结果集

函数：

  • 不可以写sql语句

  • 使用return 返回值

  • 调用时，使用函数名（）即可

  • 不能获取结果集
