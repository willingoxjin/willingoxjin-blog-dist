2.Scala 函数式编程

# 一、函数式编程

函数式编程是一种编程范式，它的主要思想是将计算过程看作是函数之间的组合。在函数式编程中，函数被视为一等公民，即函数可以被作为参数传递给其他函数、作为返回值返回、或者赋值给变量。

函数式编程具有以下特点：

  1. 纯函数：函数式编程中的函数是纯函数，即函数没有副作用，相同的输入总是产生相同的输出。
  2. 不可变性：在函数式编程中，数据是不可变的，这意味着一旦数据被创建，它就不会被修改。修改数据会创建新的数据结构，而原来的数据结构仍然存在。
  3. 高阶函数：函数式编程中的高阶函数可以接受函数作为参数，并且可以返回一个函数。
  4. 延迟计算：函数式编程中的计算通常是延迟的，即只有在需要的时候才进行计算，这可以提高程序的效率。

Scala 是一门多范式的编程语言，支持函数式编程、面向对象编程和命令式编程等多种编程范式。其中，函数式编程是 Scala 中的一个重要特性。

# 二、函数赋值给变量

Scala 中的函数是一等公民，可以独立定义，独立存在，而且可以直接将函数作为值赋值给变量。

Scala 的语法规定，将函数赋值给变量时，必须在函数后面加上空格和下划线。

示例—将函数赋值给变量：

object Person extends App {

  def sayHello(name: String): Unit = println(s"Hello! $name")

  val sayHelloFunc = sayHello _
  sayHelloFunc("zhangsan") // Hello! zhangsan

}

# 三、匿名函数

Scala 中的函数也可以不需要命名，这种函数称为匿名函数。

匿名函数的语法格式：(参数名: 参数类型) => 函数体

匿名函数可以直接赋值给变量。

使用示例：

val sayHi = (name: String) => println(s"Hi! $name")
sayHi("lisi") // Hi! lisi

val sayHiZH = (name: String) => println(s"你好! $name")
sayHiZH("王五") // 你好! 王五

# 四、高阶函数

# 1.高阶函数介绍

Scala 中函数可以直接通过参数传递到其他函数中。

接收其它函数作为当前函数的参数，当前这个函数被称作高阶函数 (higher-order function)

高阶函数使用示例：

object HigherOrderFunc extends App {

  // 定义一个匿名函数
  val sayHello = (name: String) => println(s"Hello, $name!")

  // 定义一个高阶函数
  def greeting(func: (String) => Unit, name: String): Unit = {
    func(name)
  }

  // 使用高阶函数
  greeting(sayHello, "张三") // Hello, 张三!
  greeting((name: String) => println(s"Hi, $name!"), "李四") // Hi, 李四!
  // 可以自动推断函数的参数类型
  greeting(name => println(s"你好, $name！"), "王五") // 你好, 王五！
}

# 2.常用高阶函数

Scala 中默认定义了很多高阶函数。

常用高阶函数如下：

  • map：将一个集合中的每个元素映射为一个新的元素，最终得到一个新的集合。

  • flatMap：将一个集合中的每个元素映射为一个新的集合，并将所有的新集合合并成一个新的集合。

  • foreach：对传入的每个元素都进行处理，但是没有返回值

  • filter：对传入的每个元素都进行条件判断，如果返回 true，则保留该元素；否则过滤掉该元素

  • reduce：对集合中的元素进行聚合，最终得到一个单一的值。

    • reduceLeft：从左侧元素开始，进行 reduce 操作

  • fold：对集合中的元素进行聚合，但需要提供一个初始值。

  • groupBy

  • sortBy

  • ……

使用示例：

  • map
val list = List(1, 2, 3, 4, 5)
val list2 = list.map(_ * 2)
println(list2) // List(2, 4, 6, 8, 10)

# 1.案例：统计多个文本内的单词总数

/tmp/word1.txt 文件内容：

hello hello
hi hi
cat cat cat cat
dog
rabbit rabbit rabbit
cow cow cow
1
2
3
4
5
6

/tmp/word2.txt 文件内容：

hello hello
hi hi hi
cat cat
dog dog
rabbit
cow
1
2
3
4
5
6

Scala 代码

object SumWodCount extends App {

  val words01 = scala.io.Source.fromFile("/tmp/word1.txt").mkString
  val words02 = scala.io.Source.fromFile("/tmp/word2.txt").mkString
  // 将文件内容放入List中
  val wordList = List(words01, words02)

  val sumWord = wordList.flatMap(_.split( "\n")) // 将List中的每个元素按换行符分割
    .flatMap(_.split( " ")) // 将List中的每个元素按空格分割
    .map(_.trim) // 去掉每个元素的前后空格
    .filter(_.nonEmpty) // 过滤掉空元素
    .groupBy(identity) // 按元素分组 identity 是一个函数
    .mapValues(_.size) // 计算每个单词的个数

  println(sumWord)
}
