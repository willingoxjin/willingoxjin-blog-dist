3.Spark SQL

# 一、Spark SQL 介绍

# 1.Spark SQL 是什么？

Spark SQL 是 Spark 中一个处理结构化数据的模块，它提供了与 Spark 环境集成的 API，可以使用 SQL 语言查询数据。通过 Spark SQL，你可以轻松地处理结构化数据，并可以使用 DataFrame 和 DataSet 的API 进行编程。

Spark SQL 支持多种数据源，包括 Hive 和 JSON 等，同时还支持各种数据格式，如 Parquet、Avro 和 CSV 等。如果你需要查询和处理大规模的结构化数据，Spark SQL 是一个不错的选择。

Spark SQL 和 Hive 的 Hive On Spark 是不一样的。

  • Hive On Spark是表示把底层的 MapReduce 引擎替换为 Spark 引擎。

  • Spark SQL 是 Spark 自己实现的一套SQL处理引擎。

# 2. DataFrame

Spark 中的 DataFrame 是一个分布式的数据集合，它可以看作是一张关系型数据库中的一个表或者是一个电子表格，通常由行和列组成，每一列都有一个名称和一个数据类型。

DataFrame 不同于RDD，它具有更高的抽象度，可以更方便地进行数据操作和处理，同时也可以提高处理数据的性能，因为 DataFrame 底层基于 RDD 进行实现，它可以自动优化查询计划，从而加速数据处理。

DataFrame 可以使用多种编程语言，如 Scala、Python、Java 和 R 等进行操作，非常方便。

DataFrame 是 Spark 中非常重要的部分，它为用户提供了一种更直观、更高效的数据处理方式。

Spark 1.3 出现 DataFrame ，Spark 1.6 出现了 DataSet 。在 Spark 2.0 中两者统一，DataFrame 等于 DataSet[Row]。

# 3.SparkSession

SparkSession 是 Spark SQL、DataFrame 和 DataSet 的主要入口。它是在 SparkConf 配置对象和SparkContext 上提供了一层更高级别的 API，简化了 Spark 应用程序的开发。

SparkSession 封装了 SparkConf 和 SparkContext 对象，并且提供了许多更方便的方法来方便用户处理数据。通过 SparkSession，可以快速地创建 DataFrame、DataSet、SQLContext，并且可以方便地添加资源、配置安全性等选项。如果想通过 SparkSession 来操作 RDD ，需要先通过它来获取 SparkContext。而 SqlContext 是使用SparkSQL 操作 Hive 的时候会用到的。

此外，SparkSession 还提供了大量的内置函数和连接器，支持多种数据格式、文件系统和数据源，使得数据的读取和操作更加方便和高效。

# 4.DataFrame 和 DataSet 转换

上面的 Scala 代码默认创建的是 DataFrame，Java 代码默认创建的是 DataSet。由于 DataFrame 等于DataSet[Row]，它们两个可以互相转换，所以创建哪个都是一样的。

在 Scala 代码中将 DataFrame 转换成 DataSet[Row]：

// 将DataFrame转换成DataSet[Row]
val ds = sparkSession.read.json("/tmp/users.json").as("stu")

在 Java 代码中将 DataSet[Row] 转换成 DataFrame：

// 将Dataset<Row>转换成DataFrame
Dataset<Row> df = sparkSession.read().json("/tmp/users.json").toDF();

# 二、 DataFrame 的使用

# 1.DataFrame 的常用算子

官方文档：Untyped Dataset Operations (aka DataFrame Operations) (opens new window)

常用的算子如下：

  • printSchema() 打印schema信息
  • show() 默认显示所有数据，可以通过参数控制显示多少条
  • select() 显示数据中的指定字段信息
  • filter()、where() 对数据进行过滤，需要添加隐式转换函数，否则语法报错
  • groupBy() 对数据进行分组
  • count() 计算总个数

Scala 示例代码：

object DataFrameDemo extends App {

  val conf = new SparkConf()
    .setMaster("local")

  // 创建SparkSession对象，里面包含SparkContext和SqlContext
  val sparkSession = SparkSession.builder()
    .appName("DataFrameDemo-Scala")
    .config(conf)
    .getOrCreate()

  // 读取文件，创建DataFrame
  val df = sparkSession.read.json("/tmp/users.json")
  // 打印schema信息
  df.printSchema()

  // 默认显示所有数据，可以通过参数控制显示多少条
  df.show()
  df.show(2)

  // 显示数据中的指定字段信息
  df.select("id", "name").show()
  // 在使用select的时候可以对数据做一些操作，需要添加隐式转换函数，否则语法报错
  import sparkSession.implicits._
  df.select($"id", $"name", $"age" + 1).show()

  // 对数据进行过滤，需要添加隐式转换函数，否则语法报错
  df.filter($"age" > 19).show()
  // where底层调用的就是filter
  df.where($"age" > 19).show()

  // 对数据进行分组求和
  df.groupBy("age").count().show()

  sparkSession.stop()
}

import org.apache.spark.SparkConf;
import org.apache.spark.sql.Dataset;
import org.apache.spark.sql.Row;
import org.apache.spark.sql.SparkSession;
import static org.apache.spark.sql.functions.col;

public class DataFrameDemo {

    public static void main(String[] args) {
        SparkConf conf = new SparkConf();
        conf.setMaster("local");
        SparkSession sparkSession = SparkSession.builder()
                .appName("DataFrameDemo-Java")
                .config(conf)
                .getOrCreate();
        Dataset<Row> df = sparkSession.read().json("/tmp/users.json");

        // 打印schema信息
        df.printSchema();

        // 默认显示所有数据，可以通过参数控制显示多少条
        df.show();

        df.filter(col("id").gt(2)).show();
        // where底层调用的就是filter
        df.where(col("id").gt(2)).show();

        // 对数据进行分组求和
        df.groupBy("age").count().show();

        sparkSession.stop();
    }

}

# 2.DataFrame 的 SQL 操作

Spark SQL 支持直接使用 SQL 语句。

使用 SQL 语句直接查询数据的操作步骤：

  • 先将 DataFrame 注册为一个临时表

  • 使用 SparkSession 中的 sql 函数执行 SQL 语句

Scala 示例代码：

object RunSQLQueryDemo extends App {

  val conf = new SparkConf()
    .setMaster("local")

  val sparkSession = SparkSession.builder()
    .appName("RunSQLQueryDemo-Scala")
    .config(conf)
    .getOrCreate()

  val df = sparkSession.read.json("/tmp/users.json")
  // 将DataFrame注册为一个临时表
  df.createOrReplaceTempView("user")

  // 使用sql查询临时表中的数据
  sparkSession.sql(
    """
      |SELECT *
      |FROM user
      |WHERE age >= 19
      |""".stripMargin).show()

  sparkSession.stop()

}

# 3.JOIN SQL

如果存在多个表的关联查询，可以把不同的表的数据源都读取后，再创建临时表。

除了上面 users.json 测试数据，再新建 orders.json 文件用于测试，内容如下：

{"uid":"10000001","orderNO":"PNO16823306490000001","price":"8888"}

{"uid":"10000003","orderNO":"PNO16823306490000003","price":"5800"}
{"uid":"10000004","orderNO":"PNO16823306490000004","price":"19900"}

Scala 代码如下：

object JoinSQLDemo extends App {

  val conf = new SparkConf()
    .setMaster("local")

  val sparkSession = SparkSession.builder()
    .appName("JoinSQLDemo-Scala")
    .config(conf)
    .getOrCreate()

  val userDF = sparkSession.read.json("/tmp/users.json")
  val orderDF = sparkSession.read.json("/tmp/orders.json")

  // 将DataFrame注册为一个临时表
  userDF.createOrReplaceTempView("user")
  orderDF.createOrReplaceTempView("order")

  // 查询订单金额大于等于100.00元的订单及用户信息（源数据中的单位是分）
  sparkSession.sql(
    """
      |SELECT u.id, u.name, o.orderNO, o.price
      |FROM user u
      |INNER JOIN order o ON o.uid = u.id
      |WHERE price >= 10000
      |""".stripMargin).show()

  sparkSession.stop()

}

# 三、Load/Save 函数

# 1.Load/Save 操作

对于 Spark SQL 的 DataFrame 来说，无论是从什么数据源创建出来的 DataFrame，都有一些共同的 load 和 save 操作。

  • load 操作主要用于加载数据，创建 DataFrame；
  • save 操作主要用于将 DataFrame 中的数据保存到文件中。

我们上面直接使用 json 方法加载的数据，实际上，json 方法源码中调用的 format 和 load 方法。源码如下：

@scala.annotation.varargs
def json(paths: String*): DataFrame = {
  userSpecifiedSchema.foreach(ExprUtils.checkJsonSchema(_).foreach(throw _))
  format("json").load(paths : _*)
}

5

如果不指定 format，则默认读取的数据源格式是 parquet，也可以手动指定数据源格式。Spark SQL内置了一些常见的数据源类型，比如 json、parquet、jdbc、orc、csv、text 等。

Scala 代码示例：

object LoadAndSaveDemo extends App {

  val conf = new SparkConf()
    .setMaster("local")
  val sparkSession = SparkSession.builder()
    .appName("LoadAndSaveDemo-Scala")
    .config(conf)
    .getOrCreate()

  // 读取数据
  val stuDf = sparkSession.read
    .format("json")
    .load("/tmp/student.json")

  // 保存数据
  stuDf.select("name", "age")
    .write
    .format("csv")
    .save("hdfs://hadoop01:9000/spark_test/load_and_save_demo_scala")

  sparkSession.stop()

}

# 2.Save Modes

Spark SQL 对于 save 操作，提供了不同的 Mode。

主要用来处理，当目标位置已经有数据时应该如何处理。save操作不会执行锁操作，并且也不是原子的，因此是有一定风险出现脏数据的。

SaveMode 如下：

SaveMode 说明
SaveMode.ErrorIfExists (默认) 如果目标位置已经存在数据，那么抛出一个异常
SaveMode.Append 如果目标位置已经存在数据，那么将数据追加进去
SaveMode.Overwrite 如果目标位置已经存在数据，那么就将已经存在的数据删除，用新数据进行覆盖
SaveMode.Ignore 如果目标位置已经存在数据，那么就忽略，不做任何操作

如下代码，将 SaveMode 设置为 Append，表示如果文件已存在，则将数据追加进去：

df.select("name","age")
  .write
  .format("csv")
  .mode(SaveMode.Append) // 如果文件已存在，则将数据追加进去
  .save("hdfs://hadoop01:9000/spark_test/load_and_save_demo_scala")

# 四、RDD 与 DataFrame 的转换

实际工作中，我们可能会先加载 HDFS 中的一些日志数据，然后进行一些处理，最终变成结构化的数据，再对这些数据进行统计分析。虽然我们可以使用 Spark 中提供的 Transformation 算子来实现，也可以使用 SQL 实现。不过使用 SQL 实现相对更加方便。

所以可以针对我们前面创建的 RDD，将它转换为 DataFrame，这样就可以使用 DataFrame 中的算子或者直接写 SQL 来操作数据了。

Spark SQL 支持这两种方式将 RDD 与 DataFrame 转换：

  • 反射方式
  • 编程方式

# 1.反射方式

基于反射的方式，代码比较简洁。前提是已经知道了 RDD 中的元数据，这样的话使用反射这种方式是一种非常不错的选择。

Scala 具有隐式转换的特性，所以 Spark SQL 的 Scala 接口支持自动将包含了 case class 的 RDD 转换为 DataFrame 的。

Scala 代码示例：

case class User(name: String, age: Int)

object Rdd2DataFrameByReflect extends App {

  val conf = new SparkConf()
    .setMaster("local")
  val sparkSession = SparkSession.builder()
    .appName("Rdd2DataFrameByReflect-Scala")
    .config(conf)
    .getOrCreate()
  val sc = sparkSession.sparkContext

  val dataRdd = sc.parallelize(Array(("Alice", 18), ("Bob", 25), ("Cindy", 30)))

  // 基于反射直接将包含Student对象的dataRDD转换为DataFrame
  // 需要导入隐式转换
  import sparkSession.implicits._
  val userRdd = dataRdd.map(tup => User(tup._1, tup._2))

  // RDD转换为DataFrame
  val df = userRdd.toDF()
  // 下面就可以通过DataFrame的方式操作dataRDD中的数据了
  df.createOrReplaceTempView("user")
  val resDF = sparkSession.sql("SELECT name,age FROM user WHERE age > 20")
  resDF.show()

  // 将DataFrame转化为RDD
  val resRDD = resDF.rdd
  // 从row中取数据，封装成User对象，打印到控制台
  resRDD.map(row => {
    // User(row(0).toString, row(1).toString.toInt)
    // 也可以使用row的getAs()方法，获取指定列名的值
    User(row.getAs[String]("name"), row.getAs[Int]("age"))
  }).collect()
    .foreach(println(_))

  sparkSession.stop()
}

Java 示例代码：

public class Rdd2DataFrameByReflect {

    public static class User implements Serializable {
        public static final long serialVersionUID = 1L;
        private String name;
        private Integer age;

        public User(String name, Integer age) {
            this.name = name;
            this.age = age;
        }

  			// getter setter略

        @Override
        public String toString() {
            return "User{" +
                    "name='" + name + '\'' +
                    ", age=" + age +
                    '}';
        }
    }

    public static void main(String[] args) {
        SparkConf conf = new SparkConf();
        conf.setMaster("local");

        SparkSession sparkSession = SparkSession.builder()
                .appName("Rdd2DataFrameByReflect-Java")
                .config(conf)
                .getOrCreate();

        // 从SparkSession中获取的是scala中的sparkContext，所以需要转换成java中的sparkContext
        JavaSparkContext sc = JavaSparkContext.fromSparkContext(sparkSession.sparkContext());
        Tuple2<String, Integer> user1 = new Tuple2<>("Alice", 18);
        Tuple2<String, Integer> user2 = new Tuple2<>("Bob", 25);
        Tuple2<String, Integer> user3 = new Tuple2<>("Cindy", 30);
        JavaRDD<Tuple2<String, Integer>> dataRDD = sc.parallelize(Arrays.asList(user1, user2, user3));
        // 注意：User类必须声明为public、必须实现序列化接口，且必须有getter和setter方法
        JavaRDD<User> userRDD = dataRDD.map(tup -> new User(tup._1(), tup._2()));

        // 将RDD转换成DataFrame
        Dataset<Row> df = sparkSession.createDataFrame(userRDD, User.class);
        df.createOrReplaceTempView("user");
        // 执行sql查询
        Dataset<Row> resDF = sparkSession.sql("SELECT * FROM user");
        resDF.show();

        // 将DataFrame转换成RDD
        JavaRDD<Row> resRDD = resDF.javaRDD();
        // 从row中取数据，封装成student，打印到控制台
        resRDD.map(row -> {
                    // return new User(row.getString(0), row.getInt(1));
                    return new User(row.getAs("name").toString(), row.getAs("age"));
                })
                .collect()
                .forEach(user -> System.out.println(user));

        sparkSession.stop();
    }

}

# 2.编程方式

编程方式是通过编程接口来创建 DataFrame，在程序运行时动态构建一份元数据（Schema），然后将其应用到已经存在的 RDD 上。

这种方式的代码比较冗长，但是如果在编写程序时，还不知道 RDD 的元数据，只有在程序运行时，才能动态得知其元数据，那么只能通过这种动态构建元数据的方式。即当 case class 中的字段无法预先定义的时候，就只能用编程方式动态指定元数据了

Scala 代码示例：

object Rdd2DataFrameByProgram extends App {

  val conf = new SparkConf()
    .setMaster("local")
  val sparkSession = SparkSession.builder()
    .appName("RddToDataFrameByProgramScala")
    .config(conf)
    .getOrCreate()
  val sc = sparkSession.sparkContext

  val dataRDD = sc.parallelize(Array(("Alice", 18), ("Bob", 25), ("Cindy", 30)))

  // 将RDD转化为DataFrame
  // 组装rowRDD
  val rowRDD = dataRDD.map(tup => Row(tup._1, tup._2))
  // 指定元数据信息（这个元数据信息就可以动态从外部获取）
  val schema = StructType(Array(
    StructField("name", StringType, true),
    StructField("age", IntegerType, true)
  ))
  // 构建DataFrame
  val df = sparkSession.createDataFrame(rowRDD, schema)
  df.createOrReplaceTempView("student")
  // 执行sql
  val resDf = sparkSession.sql("SELECT name,age FROM user WHERE age > 20")
  resDf.show()

  // 将DataFrame转化为RDD
  val resRDD = resDf.rdd
  resRDD.map(row => (row(0).toString, row(1).toString.toInt))
    .collect()
    .foreach(println(_))

  sparkSession.stop()

}

public class Rdd2DataFrameByProgram {

    public static void main(String[] args) {
        SparkConf conf = new SparkConf();
        conf.setMaster("local");
        SparkSession sparkSession = SparkSession.builder()
                .appName("Rdd2DataFrameByReflect-Java")
                .config(conf)
                .getOrCreate();

        // 从SparkSession中获取的是scala中的sparkContext，所以需要转换成java中的sparkContext
        JavaSparkContext sc = JavaSparkContext.fromSparkContext(sparkSession.sparkContext());
        Tuple2<String, Integer> user1 = new Tuple2<>("Alice", 18);
        Tuple2<String, Integer> user2 = new Tuple2<>("Bob", 25);
        Tuple2<String, Integer> user3 = new Tuple2<>("Cindy", 30);
        JavaRDD<Tuple2<String, Integer>> dataRDD = sc.parallelize(Arrays.asList(user1, user2, user3));

        // 将RDD转换成DataFrame
        JavaRDD<Row> rowRDD = dataRDD.map(tup -> RowFactory.create(tup._1(), tup._2()));
        // 指定元数据信息
        List<StructField> structFieldList = new ArrayList<>();
        structFieldList.add(DataTypes.createStructField("name", DataTypes.StringType, true));
        structFieldList.add(DataTypes.createStructField("age", DataTypes.IntegerType, true));
        StructType schema = DataTypes.createStructType(structFieldList);
        // 构建DataFrame
        Dataset<Row> df = sparkSession.createDataFrame(rowRDD, schema);
        df.createOrReplaceTempView("user");
        // 执行sql
        Dataset<Row> resDf = sparkSession.sql("SELECT name,age FROM user WHERE age > 20");
        resDf.show();

        // 将DataFrame转换成RDD
        JavaRDD<Row> resRDD = resDf.javaRDD();
        List<Tuple2<String, Integer>> resList = resRDD.map(row -> new Tuple2<>(row.getString(0), row.getInt(1)))
                .collect();
        resList.forEach(tup -> System.out.println(tup));

        sparkSession.stop();
    }

}
