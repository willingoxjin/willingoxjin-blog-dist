Zabbix

# 一、Zabbix 介绍

Zabbix 是一个开源的监控软件，能用来监控和跟踪各种网络服务，服务器和其他网络硬件的状态。

  • Server：Zabbix 的 Serve 端。
  • Database：Zabbix 用来存储数据数据库。
  • Agent：用来采集数据。Agent 分为两种模式：
    • 主动模式：由 Agent 主动建立 TCP 连接，并向 Server 端发送请求
    • 被动模式：由 Server 主动建立 TCP 连接，并向 Agent 端发送请求
  • Proxy：代理多台机器，多台机器将监控数据汇总后，统一发送给 Server，一般用于监控节点比较多的场景。
  • Web UI：Zabbix 的前端 Web 界面，也叫做 Zabbix frontend。

官方文档：https://www.zabbix.com/documentation/6.0/zh/manual/installation/requirements (opens new window)

dnf 是新一代的rpm软件包管理器。在 CentOS 8中，默认使用的软件管理工具由yum 换成了 速度更快的 dnf，原有的 yum 命令也成了 dnf 命令的 软链接：

~ ll /usr/bin/yum
lrwxrwxrwx. 1 root root 5  9月 22 22:33 /usr/bin/yum -> dnf-3
1
2

CentOS 8 的 DNF 源分为两个文件，分别是 CentOS-AppStream.repo 和 CentOS-Base.repo：

  • BaseOS 中的包旨在提供底层操作系统功能的核心集，为所有类型的安装提供基础。
  • 应用程序流中的包包括支持各种工作负载和用例的用户空间应用程序、语言运行库和数据库。

如果网络不好，可使用阿里云提供的镜像地址安装：

# 三、Zabbix 安装（源码方式）

由于 zabbix 在 Cent OS 7 、 ARM 架构 CPU 的机器上没有提供 RPM 包。所以只能使用源码方式安装。

官方文档：https://www.zabbix.com/documentation/6.0/zh/manual/installation/requirements (opens new window)

本示例相关环境：

  • 操作系统：CentOS 9
  • Zabbix 版本：6.0 LTS
  • 数据库：MariaDB-10.10.3
  • PHP：7.4
  • Nginx

注意：Zabbix 版本支持的数据库最高版本。见官方文档：https://www.zabbix.com/documentation/6.0/zh/manual/installation/requirements (opens new window)

关闭防火墙和安全机制：

systemctl stop firewalld.service
systemctl status firewalld.service
systemctl disable firewalld.service

setenforce 0

# 开放端口
# 开放80端口（--permanent永久生效，没有此参数重启后失效）
firewall-cmd --zone=public --add-port=80/tcp --permanent 
firewall-cmd --zone=public --add-port=3306/tcp --permanent 
# 重新载入
firewall-cmd --reload

# 查看开放状态
firewall-cmd --zone=public --query-port=80/tcp

# 1.中文字体

可在 User Setting —Profile—用户—语言修改字体：

但是 zabbix 自带的字体不支持中文，因此需要我们手动替换原来的字体文件：

可以从你自己的电脑中的字体，替换 Web 页面 assets/fonts 下的字体即可（我这里是 /usr/local/zabbix/ui/assets/fonts）

  • Windows 系统可以在 C:\Windows\Fonts 下查找 ttf 格式的字体替换
  • Mac 系统可以在字体册中找到 ttf 的字体
  • CentOS 系统字体存放在 /usr/share/fonts/ 下
  • 也可以自己下载字体
